# Prompt 1.5 — API Contract Kategori & Satuan

## Konteks
Kategori digunakan untuk mengelompokkan produk.
Satuan (units) adalah master satuan yang dipakai pada produk dan transaksi.

---

## KATEGORI

### GET /api/categories
**Auth:** Bearer Token | Semua role  

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "id": 1, "name": "Minuman", "description": "Produk minuman", "created_at": "..." }
  ]
}
```

### POST /api/categories
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{ "name": "Minuman", "description": "Produk minuman" }
```

**Validasi:** `name` required, unik

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Kategori berhasil ditambahkan", "data": { "id": 1 } }
```

### PUT /api/categories/:id
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{ "name": "Minuman Updated", "description": "..." }
```

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Kategori berhasil diupdate" }
```

### DELETE /api/categories/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus kategori yang masih dipakai produk

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Kategori berhasil dihapus" }
```

---

## SATUAN (UNITS)

### GET /api/units
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?is_active=1   → hanya yang aktif
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "id": 1, "name": "Pcs", "abbreviation": "pcs", "is_active": true }
  ]
}
```

### GET /api/units/:id
**Auth:** Bearer Token | Semua role  

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": { "id": 1, "name": "Pcs", "abbreviation": "pcs", "is_active": true }
}
```

### POST /api/units
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{ "name": "Karton", "abbreviation": "ktn" }
```

**Validasi:** `name` required, `abbreviation` required, keduanya unik

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Satuan berhasil ditambahkan", "data": { "id": 10 } }
```

### PUT /api/units/:id
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{ "name": "Karton", "abbreviation": "ktn" }
```

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Satuan berhasil diupdate" }
```

### DELETE /api/units/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus satuan yang masih dipakai produk

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Satuan berhasil dihapus" }
```

### PATCH /api/units/:id/toggle-status
**Auth:** Bearer Token | Role: owner, admin  

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Status satuan berhasil diubah", "data": { "is_active": false } }
```
