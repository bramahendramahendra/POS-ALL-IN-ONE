# Prompt 1.11 — API Contract Purchase Order

## Konteks
Purchase Order (PO) mencatat pembelian stok dari supplier.
Saat PO dibuat, stok produk bertambah otomatis.
Pembayaran bisa dilakukan parsial (dicicil) atau lunas sekaligus.

## Endpoint

### GET /api/purchases
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?supplier_id=1
?payment_status=unpaid
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "purchase_code": "PO-20240101-001",
      "supplier_name": "PT Sumber Makmur",
      "purchase_date": "2024-01-01",
      "total_amount": 1500000,
      "paid_amount": 500000,
      "remaining_amount": 1000000,
      "payment_status": "partial"
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 20 }
}
```

---

### POST /api/purchases
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "supplier_id": 1,
  "supplier_name": "PT Sumber Makmur",
  "purchase_date": "2024-01-01",
  "notes": "",
  "items": [
    {
      "product_id": 1,
      "product_name": "Kopi Susu",
      "quantity": 100,
      "unit": "pcs",
      "purchase_price": 8000,
      "subtotal": 800000
    }
  ]
}
```

**Proses di Backend:**
1. Generate `purchase_code`
2. Hitung `total_amount` dari sum items
3. Simpan PO dan items
4. Tambah stok setiap produk
5. Catat mutasi stok tipe `in`

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Purchase Order berhasil dibuat", "data": { "id": 1, "purchase_code": "PO-20240101-001" } }
```

---

### PUT /api/purchases/:id
**Auth:** Bearer Token | Role: owner, admin  

**Catatan:** Hanya bisa edit jika belum ada pembayaran

**Request Body:** Sama dengan POST

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Purchase Order berhasil diupdate" }
```

---

### DELETE /api/purchases/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus jika sudah ada pembayaran

**Proses:** Rollback stok semua item sebelum hapus

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Purchase Order berhasil dihapus" }
```

---

### POST /api/purchases/:id/pay
**Deskripsi:** Bayar PO (parsial atau lunas)  
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "amount": 500000,
  "payment_method": "cash",
  "notes": "Pembayaran tahap 1"
}
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Pembayaran berhasil dicatat",
  "data": {
    "paid_amount": 1000000,
    "remaining_amount": 500000,
    "payment_status": "partial"
  }
}
```
