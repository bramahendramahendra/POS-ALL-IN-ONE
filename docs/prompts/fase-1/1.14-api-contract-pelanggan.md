# Prompt 1.14 — API Contract Pelanggan

## Endpoint

### GET /api/customers
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?search=nama_atau_kode
?is_active=1
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "customer_code": "CUS-001",
      "name": "Budi Santoso",
      "phone": "08123456789",
      "address": "Jl. Merdeka No. 5",
      "credit_limit": 500000,
      "is_active": true
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 50 }
}
```

---

### GET /api/customers/active
**Deskripsi:** List pelanggan aktif saja (untuk dropdown di kasir)  
**Auth:** Bearer Token | Semua role  

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "id": 1, "name": "Budi Santoso", "customer_code": "CUS-001", "credit_limit": 500000 }
  ]
}
```

---

### GET /api/customers/:id
**Auth:** Bearer Token | Semua role  

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": {
    "id": 1,
    "customer_code": "CUS-001",
    "name": "Budi Santoso",
    "phone": "08123456789",
    "address": "Jl. Merdeka No. 5",
    "credit_limit": 500000,
    "notes": "",
    "is_active": true
  }
}
```

---

### POST /api/customers
**Auth:** Bearer Token | Semua role  

**Request Body:**
```json
{
  "name": "Budi Santoso",
  "phone": "08123456789",
  "address": "Jl. Merdeka No. 5",
  "credit_limit": 500000,
  "notes": ""
}
```

**Proses:** Auto-generate `customer_code` (format: CUS-001, CUS-002, dst)

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Pelanggan berhasil ditambahkan", "data": { "id": 1, "customer_code": "CUS-001" } }
```

---

### PUT /api/customers/:id
**Auth:** Bearer Token | Semua role  

**Request Body:** Sama dengan POST

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Pelanggan berhasil diupdate" }
```

---

### DELETE /api/customers/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus pelanggan yang masih punya piutang aktif

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Pelanggan berhasil dihapus" }
```

---

### PATCH /api/customers/:id/toggle-status
**Auth:** Bearer Token | Role: owner, admin  

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Status pelanggan berhasil diubah", "data": { "is_active": false } }
```
