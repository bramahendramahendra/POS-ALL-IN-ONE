# Prompt 1.20 — API Contract Laporan Stok

## Endpoint

### GET /api/reports/stock
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?category_id=1
?low_stock=1        → hanya stok di bawah min_stock
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": {
    "summary": {
      "total_products": 120,
      "total_stock_value": 45000000,
      "low_stock_count": 8
    },
    "products": [
      {
        "id": 1,
        "name": "Kopi Susu",
        "category_name": "Minuman",
        "stock": 50,
        "min_stock": 10,
        "unit": "pcs",
        "purchase_price": 8000,
        "stock_value": 400000,
        "is_low_stock": false
      }
    ],
    "pagination": { "page": 1, "limit": 20, "total": 120 }
  }
}
```

---

### GET /api/reports/stock/export
**Auth:** Bearer Token | Role: owner, admin  
**Query Params:** Sama dengan GET  

**Response:** File Excel (.xlsx) sebagai download
