# Prompt 1.22 — API Contract Dashboard

## Endpoint

### GET /api/dashboard/stats
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?date=2024-01-01    → stats untuk tanggal tertentu (default: hari ini)
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": {
    "today": {
      "total_transactions": 25,
      "total_sales": 1500000,
      "total_expenses": 50000,
      "gross_profit": 450000
    },
    "this_month": {
      "total_transactions": 450,
      "total_sales": 25000000,
      "total_expenses": 800000,
      "gross_profit": 7000000
    },
    "low_stock_count": 5,
    "open_receivables": 3
  }
}
```

### GET /api/dashboard/sales-trend
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?period=7days       → 7days, 30days, 12months
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "label": "Sen", "total_sales": 1500000, "total_transactions": 15 },
    { "label": "Sel", "total_sales": 1800000, "total_transactions": 18 }
  ]
}
```

### GET /api/dashboard/top-products
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?sort_by=quantity    → quantity atau value
?limit=10
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "product_id": 1, "product_name": "Kopi Susu", "total_qty": 250, "total_value": 3000000 }
  ]
}
```
