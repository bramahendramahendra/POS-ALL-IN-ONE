# Prompt 1.9 — API Contract Kas Harian

## Konteks
Kas harian (cash drawer) digunakan untuk mencatat saldo kas per kasir per shift.
Kasir membuka kas di awal shift dengan saldo awal, lalu menutupnya di akhir shift.
Sistem menghitung selisih antara saldo yang diharapkan dengan saldo aktual.

## Endpoint

### GET /api/cash-drawer/current
**Deskripsi:** Get kas yang sedang terbuka milik user yang login  
**Auth:** Bearer Token | Semua role  

**Response (200) — Kas Terbuka:**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": {
    "id": 1,
    "user_id": 1,
    "user_name": "Kasir Satu",
    "shift_id": 1,
    "shift_name": "Pagi",
    "open_time": "2024-01-01T08:00:00Z",
    "opening_balance": 500000,
    "total_sales": 1500000,
    "total_cash_sales": 1200000,
    "total_expenses": 50000,
    "expected_balance": 1650000,
    "status": "open"
  }
}
```

**Response (200) — Tidak Ada Kas Terbuka:**
```json
{
  "code": "00", "status": true, "message": "Tidak ada kas yang terbuka",
  "data": null
}
```

---

### POST /api/cash-drawer/open
**Deskripsi:** Buka kas baru  
**Auth:** Bearer Token | Semua role  

**Request Body:**
```json
{
  "shift_id": 1,
  "opening_balance": 500000
}
```

**Validasi:** Tidak bisa buka kas jika sudah ada kas terbuka

**Response (201):**
```json
{
  "code": "21", "status": true, "message": "Kas berhasil dibuka",
  "data": { "id": 1 }
}
```

---

### POST /api/cash-drawer/:id/close
**Deskripsi:** Tutup kas  
**Auth:** Bearer Token | Semua role  

**Request Body:**
```json
{
  "closing_balance": 1645000,
  "notes": "Selisih karena uang kembalian"
}
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Kas berhasil ditutup",
  "data": {
    "expected_balance": 1650000,
    "closing_balance": 1645000,
    "difference": -5000
  }
}
```

---

### GET /api/cash-drawer
**Deskripsi:** Riwayat kas (semua user, filter by tanggal)  
**Auth:** Bearer Token | Role: owner, admin  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?user_id=1
?status=closed
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "user_name": "Kasir Satu",
      "shift_name": "Pagi",
      "open_time": "2024-01-01T08:00:00Z",
      "close_time": "2024-01-01T16:00:00Z",
      "opening_balance": 500000,
      "closing_balance": 1645000,
      "expected_balance": 1650000,
      "difference": -5000,
      "total_sales": 1500000,
      "status": "closed"
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 30 }
}
```

---

### GET /api/cash-drawer/:id
**Auth:** Bearer Token | Semua role  

**Response (200):** Detail satu record kas (sama dengan item di list)

---

### PATCH /api/cash-drawer/:id/update-sales
**Deskripsi:** Update total penjualan di kas (dipanggil otomatis saat transaksi)  
**Auth:** Bearer Token | Internal  

**Request Body:**
```json
{
  "total_sales": 1500000,
  "total_cash_sales": 1200000
}
```

---

### PATCH /api/cash-drawer/:id/update-expenses
**Deskripsi:** Update total pengeluaran di kas (dipanggil otomatis saat input pengeluaran)  
**Auth:** Bearer Token | Internal  

**Request Body:**
```json
{ "total_expenses": 50000 }
```
