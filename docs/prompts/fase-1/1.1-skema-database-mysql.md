# Prompt 1.1 — Skema Database MySQL

## Konteks
Aplikasi POS Multi-Platform menggunakan MySQL sebagai database global.
Database ini diakses oleh Desktop (online), Web, dan Android.
Desktop juga punya SQLite lokal untuk mode offline (akan dibahas di fase 3).

Saat ini desktop menggunakan SQLite dengan 23 tabel.
Tugas ini adalah membuat skema MySQL yang setara, dengan penyesuaian untuk multi-platform.

## Tugas
Buatkan file SQL migration lengkap untuk semua tabel berikut.
Simpan di `backend/database/migrations/001_init_schema.sql`

### Tabel yang Harus Dibuat

```sql
- id             INT AUTO_INCREMENT PRIMARY KEY
- username       VARCHAR(50) UNIQUE NOT NULL

- full_name      VARCHAR(100) NOT NULL
```
