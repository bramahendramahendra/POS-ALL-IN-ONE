# Prompt 1.7 — API Contract Product Units & Harga Tier

## Konteks
Setiap produk bisa punya beberapa satuan jual alternatif dengan harga berbeda.
Contoh: Aqua bisa dijual per Pcs (Rp 5.000) atau per Dus/24pcs (Rp 100.000).
Selain itu, produk bisa punya harga tier/grosir berdasarkan minimum qty pembelian.

---

## PRODUCT UNITS (Satuan Alternatif per Produk)

### GET /api/products/:product_id/units
**Auth:** Bearer Token | Semua role  

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "product_id": 1,
      "unit_id": 2,
      "unit_name": "Dus",
      "conversion_qty": 24,
      "selling_price": 100000,
      "is_default": false
    }
  ]
}
```

---

### POST /api/products/:product_id/units
**Deskripsi:** Tambah atau update satuan alternatif (upsert)  
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "units": [
    {
      "unit_id": 2,
      "unit_name": "Dus",
      "conversion_qty": 24,
      "selling_price": 100000,
      "is_default": false
    }
  ]
}
```

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Satuan produk berhasil disimpan" }
```

---

### DELETE /api/products/:product_id/units/:unit_id
**Auth:** Bearer Token | Role: owner, admin  

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Satuan produk berhasil dihapus" }
```

---

## PRODUCT PRICES (Harga Tier/Grosir)

### GET /api/products/:product_id/prices
**Auth:** Bearer Token | Semua role  

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "product_id": 1,
      "tier_name": "Grosir",
      "min_qty": 12,
      "price": 10000
    }
  ]
}
```

---

### POST /api/products/:product_id/prices
**Deskripsi:** Simpan harga tier (replace semua)  
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "prices": [
    { "tier_name": "Grosir", "min_qty": 12, "price": 10000 },
    { "tier_name": "Partai", "min_qty": 50, "price": 9000 }
  ]
}
```

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Harga tier berhasil disimpan" }
```
