# Prompt 1.13 — API Contract Supplier

## Endpoint

### GET /api/suppliers
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?search=nama_supplier
?is_active=1
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "supplier_code": "SUP-001",
      "name": "PT Sumber Makmur",
      "phone": "021-1234567",
      "email": "info@sumbermakmur.com",
      "contact_person": "Budi",
      "is_active": true
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 10 }
}
```

---

### POST /api/suppliers
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "name": "PT Sumber Makmur",
  "address": "Jl. Raya No. 1, Jakarta",
  "phone": "021-1234567",
  "email": "info@sumbermakmur.com",
  "contact_person": "Budi",
  "notes": ""
}
```

**Proses:** Auto-generate `supplier_code` (format: SUP-001, SUP-002, dst)

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Supplier berhasil ditambahkan", "data": { "id": 1, "supplier_code": "SUP-001" } }
```

---

### PUT /api/suppliers/:id
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:** Sama dengan POST

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Supplier berhasil diupdate" }
```

---

### DELETE /api/suppliers/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus supplier yang sudah punya PO

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Supplier berhasil dihapus" }
```

---

### PATCH /api/suppliers/:id/toggle-status
**Auth:** Bearer Token | Role: owner, admin  

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Status supplier berhasil diubah", "data": { "is_active": false } }
```
