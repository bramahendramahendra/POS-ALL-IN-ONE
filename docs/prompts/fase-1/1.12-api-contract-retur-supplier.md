# Prompt 1.12 — API Contract Retur Supplier

## Konteks
Retur supplier digunakan saat ada produk yang perlu dikembalikan ke supplier.
Retur selalu berdasarkan Purchase Order yang sudah ada.
Saat retur disetujui, stok produk dikurangi otomatis.

## Endpoint

### GET /api/supplier-returns
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?supplier_id=1
?status=pending
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "return_code": "RTR-20240101-001",
      "supplier_name": "PT Sumber Makmur",
      "return_date": "2024-01-05",
      "total_return_amount": 160000,
      "status": "pending"
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 10 }
}
```

### POST /api/supplier-returns
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "purchase_id": 1,
  "supplier_id": 1,
  "supplier_name": "PT Sumber Makmur",
  "return_date": "2024-01-05",
  "reason": "Produk rusak",
  "notes": "",
  "items": [
    {
      "purchase_item_id": 1,
      "product_id": 1,
      "product_name": "Kopi Susu",
      "quantity": 20,
      "unit": "pcs",
      "purchase_price": 8000,
      "subtotal": 160000
    }
  ]
}
```

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Retur berhasil dibuat", "data": { "id": 1, "return_code": "RTR-20240101-001" } }
```

---

### PATCH /api/supplier-returns/:id/status
**Deskripsi:** Update status retur (approved/rejected)  
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{ "status": "approved", "notes": "" }
```

**Proses jika `approved`:**
- Kurangi stok setiap produk di retur
- Catat mutasi stok tipe `out`

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Status retur berhasil diupdate" }
```

---

### DELETE /api/supplier-returns/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus retur yang sudah approved

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Retur berhasil dihapus" }
```
