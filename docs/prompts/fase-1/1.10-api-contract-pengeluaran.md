# Prompt 1.10 — API Contract Pengeluaran

## Konteks
Pengeluaran (expenses) mencatat biaya operasional toko.
Setiap pengeluaran terhubung ke kas harian yang sedang terbuka.

## Endpoint

### GET /api/expenses
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?category=operasional
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "expense_date": "2024-01-01",
      "category": "Operasional",
      "description": "Beli sabun cuci",
      "amount": 25000,
      "payment_method": "cash",
      "user_name": "Admin",
      "notes": ""
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 50 }
}
```

---

### GET /api/expenses/:id
**Auth:** Bearer Token | Semua role  

**Response (200):** Detail satu pengeluaran

---

### POST /api/expenses
**Auth:** Bearer Token | Semua role  

**Request Body:**
```json
{
  "expense_date": "2024-01-01",
  "category": "Operasional",
  "description": "Beli sabun cuci",
  "amount": 25000,
  "payment_method": "cash",
  "notes": ""
}
```

**Validasi:**
- `expense_date`: required
- `category`: required
- `amount`: required, > 0

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Pengeluaran berhasil ditambahkan", "data": { "id": 1 } }
```

---

### PUT /api/expenses/:id
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:** Sama dengan POST

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Pengeluaran berhasil diupdate" }
```

---

### DELETE /api/expenses/:id
**Auth:** Bearer Token | Role: owner, admin  

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Pengeluaran berhasil dihapus" }
```
