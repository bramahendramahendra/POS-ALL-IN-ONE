# Prompt 1.16 — API Contract Shift

## Endpoint

### GET /api/shifts
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?is_active=1
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "id": 1, "name": "Pagi", "start_time": "07:00", "end_time": "15:00", "is_active": true },
    { "id": 2, "name": "Siang", "start_time": "15:00", "end_time": "22:00", "is_active": true }
  ]
}
```

### POST /api/shifts
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{ "name": "Pagi", "start_time": "07:00", "end_time": "15:00" }
```

**Validasi:**
- `name`: required, unik
- `start_time` dan `end_time`: required, format HH:MM

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Shift berhasil ditambahkan", "data": { "id": 1 } }
```

---

### PUT /api/shifts/:id
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:** Sama dengan POST

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Shift berhasil diupdate" }
```

---

### DELETE /api/shifts/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus shift yang sedang digunakan kas harian

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Shift berhasil dihapus" }
```

---

### PATCH /api/shifts/:id/toggle-status

### GET /api/shifts/summary

**Auth:** Bearer Token | Role: owner, admin  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?shift_id=1
```
