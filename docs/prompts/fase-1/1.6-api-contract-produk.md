# Prompt 1.6 — API Contract Produk

## Konteks
Produk adalah data master utama. Setiap produk punya barcode unik, stok, harga beli, dan harga jual.
Stok berubah otomatis saat transaksi, pembelian, dan retur.

## Endpoint

### GET /api/products
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?search=nama_atau_barcode
?category_id=1
?is_active=1
?low_stock=1     → hanya produk stok < min_stock
?page=1&limit=20
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    {
      "id": 1,
      "barcode": "8991234567890",
      "name": "Kopi Susu",
      "category_id": 1,
      "category_name": "Minuman",
      "purchase_price": 8000,
      "selling_price": 12000,
      "stock": 50,
      "min_stock": 10,
      "unit": "pcs",
      "is_active": true
    }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 100 }
}
```

### GET /api/products/barcode/:barcode
**Deskripsi:** Cari produk by barcode (untuk scan di kasir)  
**Auth:** Bearer Token | Semua role  

**Response (200):** Sama dengan GET by ID  
**Response (404):**
```json
{ "code": "44", "status": false, "message": "Produk tidak ditemukan" }
```

---

### GET /api/products/search
**Deskripsi:** Search produk by nama (autocomplete kasir)  
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?q=kopi&limit=10
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "id": 1, "name": "Kopi Susu", "barcode": "...", "selling_price": 12000, "stock": 50, "unit": "pcs" }
  ]
}
```

---

### POST /api/products
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:**
```json
{
  "barcode": "8991234567890",
  "name": "Kopi Susu",
  "category_id": 1,
  "purchase_price": 8000,
  "selling_price": 12000,
  "stock": 50,
  "min_stock": 10,
  "unit": "pcs"
}
```

**Validasi:**
- `name`: required
- `selling_price`: required, >= 0
- `barcode`: opsional, jika diisi harus unik

**Response (201):**
```json
{ "code": "21", "status": true, "message": "Produk berhasil ditambahkan", "data": { "id": 1 } }
```

---

### PUT /api/products/:id
**Auth:** Bearer Token | Role: owner, admin  

**Request Body:** Sama dengan POST

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Produk berhasil diupdate" }
```

---

### DELETE /api/products/:id
**Auth:** Bearer Token | Role: owner, admin  

**Validasi:** Tidak bisa hapus produk yang sudah ada di transaksi

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Produk berhasil dihapus" }
```

---

### PATCH /api/products/:id/toggle-status
**Auth:** Bearer Token | Role: owner, admin  

**Response (200):**
```json
{ "code": "00", "status": true, "message": "Status produk berhasil diubah", "data": { "is_active": false } }
```

---

### POST /api/products/import
**Deskripsi:** Import produk dari file Excel/CSV  
**Auth:** Bearer Token | Role: owner, admin  
**Content-Type:** multipart/form-data  

**Form Data:**
```
file: [file Excel/CSV]
```

**Format kolom Excel:**
```
barcode | name | category | purchase_price | selling_price | stock | min_stock | unit
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Import selesai",
  "data": {
    "success": 45,
    "failed": 2,
    "errors": [
      { "row": 3, "message": "Barcode duplikat" }
    ]
  }
}
```
