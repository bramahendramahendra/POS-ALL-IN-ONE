# Prompt 1.21 — API Contract Laporan Kasir

## Endpoint

### GET /api/reports/cashier
**Auth:** Bearer Token | Role: owner, admin  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?user_id=1
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": {
    "period": { "start": "2024-01-01", "end": "2024-01-31" },
    "cashiers": [
      {
        "user_id": 1,
        "user_name": "Kasir Satu",
        "total_transactions": 45,
        "total_sales": 2500000,
        "total_cash": 1800000,
        "total_non_cash": 700000,
        "avg_transaction": 55556
      }
    ]
  }
}
```

---

### GET /api/reports/cashier/export
**Auth:** Bearer Token | Role: owner, admin  
**Query Params:** Sama dengan GET  

**Response:** File Excel (.xlsx) sebagai download
