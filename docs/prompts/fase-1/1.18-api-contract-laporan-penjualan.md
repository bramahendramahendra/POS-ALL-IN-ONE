# Prompt 1.18 — API Contract Laporan Penjualan

## Endpoint

### GET /api/reports/sales
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?user_id=1
?payment_method=cash
?page=1&limit=20
```

**Response (200):**
```json
{

  "data": {
    "summary": {
      "total_transactions": 150,
      "total_sales": 15000000,
      "total_discount": 500000,
      "total_tax": 0,
      "net_sales": 14500000

    "transactions": [
      {
        "id": 1,
```

---

### GET /api/reports/sales/chart
**Deskripsi:** Data chart penjualan per hari/minggu/bulan  
**Auth:** Bearer Token | Semua role  

**Query Params:**
```
?start_date=2024-01-01
?end_date=2024-01-31
?period=daily        → daily, weekly, monthly
```

**Response (200):**
```json
{
  "code": "00", "status": true, "message": "Success",
  "data": [
    { "label": "2024-01-01", "total_sales": 1500000, "total_transactions": 15 },
    { "label": "2024-01-02", "total_sales": 1800000, "total_transactions": 18 }
  ]
}
```

---

### GET /api/reports/sales/export
**Deskripsi:** Export laporan penjualan ke Excel  
**Auth:** Bearer Token | Role: owner, admin  
**Query Params:** Sama dengan GET /api/reports/sales  

**Response:** File Excel (.xlsx) sebagai download
