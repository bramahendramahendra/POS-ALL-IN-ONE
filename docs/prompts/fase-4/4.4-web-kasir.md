# Prompt 4.4 — Web Halaman Kasir

## File
`web/kasir.html`, `web/src/js/kasir.js`

## Instruksi
Port halaman `desktop/src/kasir.html` ke web dengan ketentuan:

1. **Salin struktur HTML** dari `desktop/src/kasir.html`
2. **Salin JS** dari `desktop/src/js/kasir.js`
3. **Pastikan** `device_source: 'web'` dikirim pada setiap create transaksi:

```javascript
// kasir.js
await apiClient.post('/transactions', {
    ...transactionData,
    device_source: 'web', // prefix kode: WEB-YYYYMMDD-XXX
});
```

4. **Hapus** semua logika offline (SQLite, sync_queue) — web tidak memiliki mode offline
5. **Cetak struk** menggunakan `window.print()` dengan CSS print-only:

```javascript
function printReceipt(transaction) {
    const printArea = document.getElementById('receipt-print');
    printArea.innerHTML = generateReceiptHTML(transaction);
    window.print();
}
```

```css
@media print {
    body > *:not(#receipt-print) { display: none; }
    #receipt-print { display: block; }
}
```

6. **Tambahkan** guard auth:
```javascript
requireAuth();
requireRole('kasir', 'owner', 'admin');
```

## Perbedaan dengan Desktop
| Fitur | Desktop | Web |
|---|---|---|
| Mode offline | Ada | Tidak ada |
| device_source | `desktop` | `web` |
| Kode transaksi | DSK-YYYYMMDD-XXX | WEB-YYYYMMDD-XXX |
| Cetak struk | Thermal/print dialog | Browser print dialog |

## Output yang Diharapkan
Kasir web berfungsi penuh: scan barcode, tambah item, proses pembayaran, cetak struk.
