# Prompt 4.3 — Web Halaman Dashboard

## File
`web/dashboard.html`, `web/src/js/dashboard.js`

## Instruksi
Port halaman `desktop/src/dashboard.html` ke web dengan ketentuan:

1. **Salin struktur HTML** dari `desktop/src/dashboard.html`
2. **Salin JS** dari `desktop/src/js/dashboard.js`
3. **Hapus** semua referensi `window.api.*` atau `window.electronAPI.*` jika masih ada
4. **Pastikan** semua API call menggunakan `apiClient.*` (sudah dilakukan di fase 3.18)
5. **Tambahkan** guard auth di awal:

```javascript
// dashboard.js — baris pertama
requireAuth();
```

6. **Sesuaikan** navbar/sidebar untuk web:
   - Tidak ada menu PIN Lock
   - Tambahkan tombol Logout di navbar
   - Sync Center hanya tampil untuk role `owner` dan `admin`

## Perbedaan dengan Desktop
- Tidak ada mode offline
- Tidak ada indikator koneksi SQLite
- Navigasi antar halaman via `window.location.href` bukan Electron IPC

## Output yang Diharapkan
Halaman dashboard web menampilkan statistik, trend penjualan, dan top produk via API.
