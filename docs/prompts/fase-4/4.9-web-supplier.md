# Prompt 4.9 — Web Halaman Supplier

## File
`web/suppliers.html`, `web/src/js/suppliers.js`

## Instruksi
Port halaman supplier dari desktop ke web:

1. **Salin** HTML dan JS dari `desktop/src/suppliers.html`
2. **Tambahkan** guard auth:

```javascript
requireAuth();
requireRole('owner', 'admin');
```

3. Halaman ini mencakup tab **Supplier** dan **Retur Supplier**
4. Tidak ada perubahan logic

## Output yang Diharapkan
Halaman supplier dan retur supplier web berfungsi penuh.
