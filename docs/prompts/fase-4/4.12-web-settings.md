# Prompt 4.12 — Web Halaman Settings

## File
`web/settings.html`, `web/src/js/settings.js`

## Instruksi
Port halaman settings dari desktop ke web:

1. **Salin** HTML dan JS dari `desktop/src/settings.html`
2. **Tambahkan** guard auth:

```javascript
requireAuth();
requireRole('admin'); // hanya admin yang bisa ubah settings
```

3. **Hapus** bagian pengaturan yang spesifik desktop:
   - Pengaturan printer thermal lokal
   - Pengaturan path backup lokal
   - Pengaturan PIN Lock

4. **Tambahkan** bagian pengaturan web:
   - URL Backend API (readonly, tampilkan saja)
   - Nama toko, pajak, dll (tetap ada)

## Output yang Diharapkan
Halaman settings web berfungsi untuk pengaturan toko umum.
