# Prompt 4.8 — Web Halaman Pelanggan & Piutang

## File
`web/customers.html`, `web/receivables.html`, `web/src/js/customers.js`, `web/src/js/receivables.js`

## Instruksi
Port halaman pelanggan dan piutang dari desktop ke web:

1. **Salin** HTML dan JS dari `desktop/src/customers.html` dan `desktop/src/receivables.html`
2. **Tambahkan** guard auth di masing-masing file JS:

```javascript
// customers.js
requireAuth();
requireRole('owner', 'admin');

// receivables.js
requireAuth();
requireRole('kasir', 'owner', 'admin'); // kasir bisa lihat piutang pelanggan
```

3. Tidak ada perubahan logic — semua API call sudah menggunakan `apiClient.*`

## Output yang Diharapkan
Halaman pelanggan dan piutang web berfungsi penuh.
