# Prompt 4.5 — Web Halaman Produk

## File
`web/products.html`, `web/src/js/products.js`

## Instruksi
Port halaman `desktop/src/products.html` ke web:

1. **Salin** HTML dan JS dari desktop
2. **Import Bulk** — gunakan `<input type="file">` HTML (sudah direfactor di 3.7, tidak perlu ubah)
3. **Cetak barcode** — gunakan `window.print()` (sudah direfactor di 3.21)
4. **Tambahkan** guard:

```javascript
requireAuth();
requireRole('owner', 'admin');
```

## Tidak Ada Perubahan Logic
Semua API call sudah menggunakan `apiClient.*` sejak fase 3. Cukup copy file dan tambahkan guard auth.

## Output yang Diharapkan
Halaman produk web berfungsi: CRUD produk, import bulk Excel, cetak label barcode.
