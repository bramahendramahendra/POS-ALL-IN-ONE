# Prompt 4.11 — Web Halaman Laporan

## File
`web/reports.html`, `web/src/js/reports.js`

## Instruksi
Port halaman laporan dari desktop ke web:

1. **Salin** HTML dan JS dari `desktop/src/reports.html`
2. **Tambahkan** guard auth:

```javascript
requireAuth();
requireRole('owner', 'admin');
```

3. **Export Excel** tetap menggunakan blob download (sudah direfactor di 3.17) — tidak ada perubahan
4. Laporan mencakup tab: Penjualan, Laba Rugi, Stok, Kasir

## Output yang Diharapkan
Semua tab laporan web berfungsi dengan filter dan export Excel via download.
