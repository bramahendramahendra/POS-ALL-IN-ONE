# Prompt 4.10 — Web Halaman Shift

## File
`web/shifts.html`, `web/src/js/shifts.js`

## Instruksi
Port halaman shift dari desktop ke web:

1. **Salin** HTML dan JS dari `desktop/src/shifts.html`
2. **Tambahkan** guard auth:

```javascript
requireAuth();
requireRole('kasir', 'owner', 'admin');
```

3. Tidak ada perubahan logic

## Output yang Diharapkan
Halaman shift web berfungsi: start/end shift, riwayat shift, summary shift.
