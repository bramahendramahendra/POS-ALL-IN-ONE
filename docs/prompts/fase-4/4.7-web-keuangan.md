# Prompt 4.7 — Web Halaman Keuangan

## File
`web/finance.html`, `web/src/js/finance.js`

## Halaman Ini Mencakup (Tab)
- Kas Harian
- Pengeluaran
- Purchase Order (Pembelian)

## Instruksi
Port halaman `desktop/src/finance.html` ke web:

1. **Salin** HTML dan JS dari desktop
2. **Tambahkan** guard auth:

```javascript
requireAuth();
requireRole('owner', 'admin');
```

3. **Hapus** semua logika offline dari finance.js — pengeluaran dan kas harian tidak bisa dilakukan offline di web
4. **Purchase Order** — upload invoice jika ada tetap menggunakan `<input type="file">` HTML

## Output yang Diharapkan
Halaman keuangan web berfungsi: kas harian, pengeluaran, dan purchase order.
