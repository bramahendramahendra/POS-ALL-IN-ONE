# Prompt 4.6 — Web Halaman Transaksi

## File
`web/transactions.html`, `web/src/js/transactions.js`

## Instruksi
Port halaman riwayat transaksi dari desktop ke web:

1. **Salin** HTML dan JS dari `desktop/src/transactions.html`
2. **Tambahkan** guard auth:

```javascript
requireAuth();
requireRole('kasir', 'owner', 'admin');
```

3. **Filter device_source** — tambahkan filter "Semua Platform" di dropdown:

```javascript
const transactions = await apiClient.get('/transactions', {
    start_date, end_date, payment_method,
    device_source, // 'desktop' | 'web' | 'android' | kosong (semua)
    page, limit,
});
```

4. **Void transaksi** tetap memerlukan role `owner` atau `admin`

## Output yang Diharapkan
Halaman riwayat transaksi web berfungsi dengan filter lintas platform (desktop/web/android).
