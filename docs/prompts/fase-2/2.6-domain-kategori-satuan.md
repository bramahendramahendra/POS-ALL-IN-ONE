# Prompt 2.6 — Domain Kategori & Satuan

## Konteks
Dua domain kecil tapi penting sebagai master data.
Bisa digabung dalam satu domain `master` atau dibuat terpisah `category` dan `unit`.

## Service Logic — Validasi
```go
// Category
func (s *categoryService) Delete(id int) error {
    // Cek apakah kategori masih dipakai produk
    count, _ := s.repo.CountProductsByCategory(id)
    if count > 0 {
        return &errors.BadRequestError{Message: "Kategori masih digunakan oleh produk"}
    }
    return s.repo.Delete(id)
}

// Unit
func (s *unitService) Delete(id int) error {
    // Cek apakah satuan masih dipakai product_units
    count, _ := s.repo.CountProductUnitsByUnit(id)
    if count > 0 {
        return &errors.BadRequestError{Message: "Satuan masih digunakan oleh produk"}
    }
    return s.repo.Delete(id)
}
```

## Routes
```go
// Kategori — semua role bisa read, owner/admin bisa write
categoryGroup := protected.Group("/categories")
categoryGroup.GET("", categoryHandler.GetAll)
categoryGroup.GET("/:id", categoryHandler.GetByID)
categoryGroup.POST("", RoleMiddleware("owner","admin"), categoryHandler.Create)
categoryGroup.PUT("/:id", RoleMiddleware("owner","admin"), categoryHandler.Update)
categoryGroup.DELETE("/:id", RoleMiddleware("owner","admin"), categoryHandler.Delete)

// Satuan
unitGroup := protected.Group("/units")
unitGroup.GET("", unitHandler.GetAll)
unitGroup.GET("/active", unitHandler.GetActive)
unitGroup.GET("/:id", unitHandler.GetByID)
unitGroup.POST("", RoleMiddleware("owner","admin"), unitHandler.Create)
unitGroup.PUT("/:id", RoleMiddleware("owner","admin"), unitHandler.Update)
unitGroup.DELETE("/:id", RoleMiddleware("owner","admin"), unitHandler.Delete)
unitGroup.PATCH("/:id/toggle-status", RoleMiddleware("owner","admin"), unitHandler.ToggleStatus)
```
