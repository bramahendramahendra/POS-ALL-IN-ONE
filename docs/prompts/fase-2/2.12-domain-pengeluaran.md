## DTO
```go
type ExpenseRequest struct {
    ExpenseDate   string  `json:"expense_date" validate:"required,dateformat=2006-01-02"`
    Category      string  `json:"category" validate:"required"`
    Description   string  `json:"description"`
    Amount        float64 `json:"amount" validate:"required,gt=0"`
    PaymentMethod string  `json:"payment_method" validate:"required,oneof=cash debit credit qris"`
    Notes         string  `json:"notes"`
}
```

## Routes
```go
expenseGroup := protected.Group("/expenses")
expenseGroup.GET("", expenseHandler.GetAll)
expenseGroup.GET("/:id", expenseHandler.GetByID)
expenseGroup.POST("", expenseHandler.Create)
expenseGroup.PUT("/:id", RoleMiddleware("owner","admin"), expenseHandler.Update)
expenseGroup.DELETE("/:id", RoleMiddleware("owner","admin"), expenseHandler.Delete)
```
