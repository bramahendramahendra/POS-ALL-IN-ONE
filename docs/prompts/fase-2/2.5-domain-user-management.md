# Prompt 2.5 — Domain User Management

## Konteks
CRUD user dengan role-based access. Hanya owner & admin yang bisa kelola user.
Ikuti pola domain: handler → service → repo → model → dto.

## Struktur
```
domain/user/
├── handler/handler_user.go
├── service/service_user_interface.go
├── service/service_user.go
├── repo/repo_user_interface.go
├── repo/repo_user.go
├── model/model_user.go
└── dto/dto_user.go
```

## Model
```go
type User struct {
    ID        int       `gorm:"column:id"`
    Username  string    `gorm:"column:username"`
    Password  string    `gorm:"column:password"`
    FullName  string    `gorm:"column:full_name"`
    Role      string    `gorm:"column:role"`
    IsActive  bool      `gorm:"column:is_active"`
    CreatedAt time.Time `gorm:"column:created_at"`
    UpdatedAt time.Time `gorm:"column:updated_at"`
}
```

## DTO
```go
type CreateUserRequest struct {
    Username string `json:"username" validate:"required,min=3,alphanum"`
    Password string `json:"password" validate:"required,min=6"`
    FullName string `json:"full_name" validate:"required"`
    Role     string `json:"role" validate:"required,oneof=owner admin kasir"`
}

type UpdateUserRequest struct {
    FullName string `json:"full_name" validate:"required"`
    Role     string `json:"role" validate:"required,oneof=owner admin kasir"`
    Password string `json:"password" validate:"omitempty,min=6"`
}

type UserResponse struct {
    ID        int       `json:"id"`
    Username  string    `json:"username"`
    FullName  string    `json:"full_name"`
    Role      string    `json:"role"`
    IsActive  bool      `json:"is_active"`
    CreatedAt time.Time `json:"created_at"`
}
```

## Query Params Support
- `?search=` → filter by username atau full_name (LIKE)
- `?role=` → filter by role
- `?is_active=` → filter by status
