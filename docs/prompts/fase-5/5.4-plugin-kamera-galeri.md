# Prompt 5.4 — Plugin Kamera & Galeri Android

## Kegunaan
- Foto produk (upload gambar produk)
- Scan barcode via kamera

## Library
```bash
npm install @capacitor/camera
npx cap sync android
```

## Implementasi: Foto Produk

```javascript
// android/src/js/camera.js
import { Camera, CameraResultType, CameraSource } from '@capacitor/camera';

async function pickProductImage() {
    const image = await Camera.getPhoto({
        quality: 80,
        allowEditing: false,
        resultType: CameraResultType.Blob,
        source: CameraSource.Prompt, // pilih: kamera atau galeri
    });

    // Konversi ke File untuk upload
    const blob = image.blob;
    const file = new File([blob], `product-${Date.now()}.jpg`, { type: 'image/jpeg' });

    // Upload ke backend
    const formData = new FormData();
    formData.append('image', file);
    const result = await apiClient.uploadFile('/products/upload-image', formData);
    return result.image_url;
}
```

## Implementasi: Scan Barcode

```javascript
// Gunakan library ZXing via web (tidak perlu plugin native)
// Karena Capacitor menjalankan web app, bisa pakai @zxing/browser

import { BrowserMultiFormatReader } from '@zxing/browser';

async function scanBarcode() {
    const codeReader = new BrowserMultiFormatReader();
    const videoEl = document.getElementById('barcode-scanner-video');

    return new Promise((resolve, reject) => {
        codeReader.decodeFromVideoDevice(null, videoEl, (result, err) => {
            if (result) {
                codeReader.reset();
                resolve(result.getText());
            }
        });
    });
}
```

## Penggunaan di products.js

```javascript
// Tombol "Ambil Foto"
document.getElementById('btn-take-photo').addEventListener('click', async () => {
    if (window.APP_CONFIG?.platform === 'android') {
        const imageUrl = await pickProductImage();
        document.getElementById('product-image-preview').src = imageUrl;
        document.getElementById('product-image-url').value = imageUrl;
    } else {
        // Desktop/Web: gunakan input file HTML
        document.getElementById('product-image-input').click();
    }
});
```

## Output yang Diharapkan
Upload foto produk via kamera/galeri di Android berfungsi, scan barcode produk berfungsi via kamera.
