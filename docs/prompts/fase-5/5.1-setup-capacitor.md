# Prompt 5.1 — Setup Capacitor

## Prasyarat
- Fase 4 (Web App) sudah selesai
- Node.js 20+ terinstall

## Inisialisasi Capacitor di folder `android/`

```bash
# Di root Project_POS
mkdir android && cd android

# Init project web untuk Capacitor (salin dari web/)
cp -r ../web/* .

# Install Capacitor
npm init -y
npm install @capacitor/core @capacitor/cli @capacitor/android

# Inisialisasi Capacitor
npx cap init "POS System" "com.yourcompany.possystem" --web-dir "."
```

## File: capacitor.config.json

```json
{
  "appId": "com.yourcompany.possystem",
  "appName": "POS System",
  "webDir": ".",
  "server": {
    "androidScheme": "https"
  },
  "plugins": {
    "SplashScreen": {
      "launchShowDuration": 2000,
      "backgroundColor": "#1e40af",
      "showSpinner": false
    }
  }
}
```

## Tambah Platform Android

```bash
npx cap add android
```

## Sesuaikan api-client.js untuk Android

```javascript
// android/src/js/app.js
window.APP_CONFIG = {
    backendUrl: 'https://api.namadomain.com', // selalu online
    platform: 'android',
};
```

## Pastikan `device_source: 'android'` di kasir.js

```javascript
await apiClient.post('/transactions', {
    ...transactionData,
    device_source: 'android', // prefix kode: AND-YYYYMMDD-XXX
});
```

## Sync setelah perubahan file web

```bash
npx cap sync android
```
