# Prompt 3.17 — Refactor Laporan Desktop

## File yang Diubah
`desktop/src/js/reports.js`, `desktop/main.js`, `desktop/preload.js`

## Perubahan

```javascript
// SEBELUM
const salesReport = await window.api.reports.getSales(filters);
const profitReport = await window.api.reports.getProfitLoss(filters);
const stockReport = await window.api.reports.getStock(filters);
const cashierReport = await window.api.reports.getCashier(filters);
await window.api.reports.exportExcel(type, filters);

// SESUDAH
const salesReport = await apiClient.get('/reports/sales', { start_date, end_date, page, limit });
const profitReport = await apiClient.get('/reports/profit-loss', { start_date, end_date });
const stockReport = await apiClient.get('/reports/stock', { category_id, is_low_stock, page, limit });
const cashierReport = await apiClient.get('/reports/cashier', { start_date, end_date, user_id });

// Export Excel — download file
const blob = await apiClient.get(`/reports/${type}/export`, filters, { responseType: 'blob' });
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = `laporan-${type}-${Date.now()}.xlsx`;
a.click();
URL.revokeObjectURL(url);
```

## Hapus dari main.js & preload.js
- `reports:*` handlers

## Catatan
- Export Excel mengembalikan file blob, bukan path file lokal
- Gunakan URL.createObjectURL untuk trigger download di browser/Electron

## Output yang Diharapkan
Semua halaman laporan berfungsi via API backend, export Excel via download browser.
