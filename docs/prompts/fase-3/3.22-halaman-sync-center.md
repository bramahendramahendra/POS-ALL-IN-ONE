# Prompt 3.22 — Halaman Sync Center Desktop

## File Baru
`desktop/src/sync-center.html`, `desktop/src/js/sync-center.js`

## Akses
- Menu sidebar: **Sync Center** (hanya tampil untuk role `owner` dan `admin`)
- Ikon: ikon sync / cloud-upload

## Struktur Halaman

### Tab 1 — Konflik Transaksi
Menampilkan daftar hari yang memiliki konflik antara data desktop (offline) vs data online.

```javascript
// Load konflik
const conflicts = await apiClient.get('/sync/conflicts', { status: 'pending', page, limit });

// Tampilkan per tanggal:
// [ 2024-01-15 ] — 3 konflik transaksi
//   [Lihat Detail] [Approve Semua] [Reject Semua]

// Approve satu konflik
await apiClient.post(`/sync/conflicts/${conflictId}/resolve`, { action: 'approve' });

// Reject satu konflik
await apiClient.post(`/sync/conflicts/${conflictId}/resolve`, { action: 'reject' });
```

**Tampilan detail konflik:**
| Field | Versi Desktop | Versi Online |
|-------|--------------|--------------|
| Total | Rp 150.000 | Rp 145.000 |
| Item | 3 item | 3 item |
| Waktu | 10:32:05 | 10:32:11 |

Tombol: **[✓ Gunakan Desktop]** | **[✗ Gunakan Online]**

---

### Tab 2 — Antrian Sync
Menampilkan antrian item yang menunggu sinkronisasi.

```javascript
// Load antrian
const queue = await apiClient.get('/sync/queue', { status, page, limit });

// Tampilkan status badge:
// PENDING  → kuning
// SYNCING  → biru (animasi)
// SYNCED   → hijau
// FAILED   → merah

// Retry item gagal
await apiClient.post(`/sync/queue/${itemId}/retry`);

// Trigger sync manual
await apiClient.post('/sync/push');
```

---

### Tab 3 — Riwayat Sync
```javascript
const history = await apiClient.get('/sync/history', { start_date, end_date, page, limit });

```

## Indikator Status Koneksi
Tampilkan di pojok kanan atas:
- 🟢 **Online** — koneksi ke backend aktif
- 🔴 **Offline** — tidak ada koneksi

## Output yang Diharapkan
Halaman Sync Center berfungsi lengkap: konflik, antrian, riwayat, dan indikator koneksi.
