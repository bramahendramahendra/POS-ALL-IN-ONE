# Prompt 3.25 — Antrian Sync: Struktur Tabel

## File Diubah
`desktop/main.js` (inisialisasi tabel), skema SQLite lokal

## Tabel sync_queue di SQLite

```sql
CREATE TABLE IF NOT EXISTS sync_queue (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    entity      TEXT NOT NULL,      -- 'transaction' | 'expense' | 'cash_drawer'
    action      TEXT NOT NULL,      -- 'create' | 'update' | 'delete'
    local_id    TEXT,               -- ID lokal (UUID) jika belum punya server ID
    server_id   INTEGER,            -- ID di server jika sudah ada
    payload     TEXT NOT NULL,      -- JSON data yang akan dikirim ke backend
    status      TEXT NOT NULL DEFAULT 'PENDING',
                                    -- PENDING | SYNCING | SYNCED | FAILED
    retry_count INTEGER DEFAULT 0,
    error_msg   TEXT,               -- Pesan error jika FAILED
    created_at  TEXT NOT NULL,
    updated_at  TEXT NOT NULL
);

CREATE INDEX IF NOT EXISTS idx_sync_queue_status   ON sync_queue(status);
CREATE INDEX IF NOT EXISTS idx_sync_queue_entity   ON sync_queue(entity);
CREATE INDEX IF NOT EXISTS idx_sync_queue_created  ON sync_queue(created_at);
```

## Status Lifecycle

```
PENDING → SYNCING → SYNCED
                  ↘ FAILED (retry_count < 3) → PENDING
                           (retry_count >= 3) → tetap FAILED, butuh resolve manual
```

## IPC Handler di main.js

```javascript
// Tambah item ke antrian
ipcMain.handle('syncQueue:add', async (event, { entity, action, localId, serverId, payload }) => {
    const now = new Date().toISOString();
    const result = db.prepare(`
        INSERT INTO sync_queue (entity, action, local_id, server_id, payload, status, created_at, updated_at)
        VALUES (?, ?, ?, ?, ?, 'PENDING', ?, ?)
    `).run(entity, action, localId || null, serverId || null, JSON.stringify(payload), now, now);
    return result.lastInsertRowid;
});

// Ambil semua item PENDING
ipcMain.handle('syncQueue:getPending', async () => {
    return db.prepare(`
        SELECT * FROM sync_queue WHERE status = 'PENDING' ORDER BY created_at ASC
    `).all();
});

// Update status item
ipcMain.handle('syncQueue:updateStatus', async (event, { id, status, errorMsg }) => {
    db.prepare(`
        UPDATE sync_queue SET status = ?, error_msg = ?, updated_at = ? WHERE id = ?
    `).run(status, errorMsg || null, new Date().toISOString(), id);
    return true;
});

// Ambil semua antrian (untuk halaman Sync Center)
ipcMain.handle('syncQueue:getAll', async (event, { status, limit, offset }) => {
    const where = status ? `WHERE status = '${status}'` : '';
    return db.prepare(`SELECT * FROM sync_queue ${where} ORDER BY created_at DESC LIMIT ? OFFSET ?`)
             .all(limit || 50, offset || 0);
});
```
