# Prompt 3.5 — Refactor Kategori & Satuan Desktop

## File yang Diubah
`desktop/src/js/products.js` (bagian kategori & satuan), `desktop/main.js`, `desktop/preload.js`

## Perubahan

### Kategori
```javascript
// SEBELUM
const categories = await window.api.categories.getAll();
await window.api.categories.create(data);
await window.api.categories.update(id, data);
await window.api.categories.delete(id);

// SESUDAH
const categories = await apiClient.get('/categories');
await apiClient.post('/categories', data);
await apiClient.put(`/categories/${id}`, data);
await apiClient.delete(`/categories/${id}`);
```

### Satuan
```javascript
// SEBELUM
const units = await window.api.units.getAll();
const activeUnits = await window.api.units.getActive();
await window.api.units.create(data);
await window.api.units.update(id, data);
await window.api.units.delete(id);
await window.api.units.toggleStatus(id);

// SESUDAH
const units = await apiClient.get('/units');
const activeUnits = await apiClient.get('/units/active');
await apiClient.post('/units', data);
await apiClient.put(`/units/${id}`, data);
await apiClient.delete(`/units/${id}`);
await apiClient.patch(`/units/${id}/toggle-status`);
```

## Hapus dari main.js & preload.js
- `categories:*` handlers
- `units:*` handlers

## Output yang Diharapkan
Tab kategori dan satuan di halaman produk berfungsi via API backend.
