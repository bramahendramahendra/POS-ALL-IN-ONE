# Prompt 3.20 — Refactor Backup & Restore Desktop

## File yang Diubah
`desktop/src/js/backup.js`, `desktop/main.js`, `desktop/preload.js`

## Perubahan

```javascript
// SEBELUM
const result = await window.api.backup.export();       // simpan file lokal via dialog
await window.api.backup.restore(filePath);             // baca file lokal

// SESUDAH

// Export — download file dari backend
const blob = await apiClient.get('/backup/export', {}, { responseType: 'blob' });
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = `backup-${new Date().toISOString().slice(0,10)}.sql`;
a.click();
URL.revokeObjectURL(url);

// Restore — upload file ke backend
const formData = new FormData();
formData.append('file', fileInput.files[0]);
await apiClient.uploadFile('/backup/restore', formData);
```

## Hapus dari main.js & preload.js
- `backup:*` handlers
- Hapus penggunaan `dialog.showSaveDialog` dan `dialog.showOpenDialog` untuk backup

## Catatan
- Export menggunakan blob download, tidak lagi menyimpan ke path lokal via Electron dialog
- Restore menggunakan file input HTML `<input type="file" accept=".sql,.zip">`, bukan dialog Electron
- Konfirmasi restore tetap ditampilkan via modal sebelum proses

## Output yang Diharapkan
Backup download otomatis via browser, restore via upload file — tidak bergantung dialog Electron.
