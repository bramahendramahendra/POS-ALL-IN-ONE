# Prompt 3.4 — Refactor User Management Desktop

## File yang Diubah
`desktop/src/js/users.js`, `desktop/src/views/users.html`

## Pola Refactor (ikuti untuk semua file berikutnya)
Ganti semua `window.api.[module].[method]()` dengan `apiClient.[method]('/endpoint', data)`.

## Perubahan di `users.js`

### Sebelum → Sesudah
```javascript
// SEBELUM
const users = await window.api.users.getAll();
const user = await window.api.users.getById(id);
await window.api.users.create(data);
await window.api.users.update(id, data);
await window.api.users.delete(id);
await window.api.users.toggleStatus(id);

// SESUDAH
const users = await apiClient.get('/users', { search, role, is_active });
const user = await apiClient.get(`/users/${id}`);
await apiClient.post('/users', data);
await apiClient.put(`/users/${id}`, data);
await apiClient.delete(`/users/${id}`);
await apiClient.patch(`/users/${id}/toggle-status`);
```

## Hapus dari `desktop/main.js`
Hapus semua IPC handlers: `users:getAll`, `users:getById`, `users:create`, `users:update`, `users:delete`, `users:toggleStatus`

## Hapus dari `desktop/preload.js`
Hapus expose `users.*`

## Output yang Diharapkan
Halaman manajemen user berfungsi via API backend.
