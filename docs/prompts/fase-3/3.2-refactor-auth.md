# Prompt 3.2 — Refactor Auth Desktop

## Konteks
File `desktop/src/js/auth.js` saat ini menggunakan `window.api.auth.*` (IPC).
Ganti semua IPC calls dengan `apiClient` dari prompt 3.1.

## File yang Diubah
`desktop/src/js/auth.js`

## Perubahan

### Login (Sebelum → Sesudah)
```javascript
// SEBELUM (IPC)
async function login(username, password) {
    const result = await window.api.auth.login({ username, password });
    // ...
}

// SESUDAH (fetch ke backend)
async function login(username, password) {
    try {
        const data = await apiClient.post('/auth/login', {
            username,
            password,
            device_info: 'desktop'
        });

        // Simpan token dan user ke localStorage
        apiClient.setToken(data.token);
        localStorage.setItem('refresh_token', data.refresh_token);
        localStorage.setItem('user', JSON.stringify(data.user));

        // Redirect ke dashboard
        window.location.href = 'dashboard.html';
    } catch (error) {
        showError(error.message);
    }
}
```

### Logout (Sebelum → Sesudah)
```javascript
// SEBELUM
await window.api.auth.logout();

// SESUDAH
async function logout() {
    try {
        await apiClient.post('/auth/logout');
    } finally {
        apiClient.clearToken();
        window.location.href = 'login.html';
    }
}
```

### Get Current User (Sebelum → Sesudah)
```javascript
// SEBELUM
const user = await window.api.auth.getCurrentUser();

// SESUDAH
function getCurrentUser() {
    return JSON.parse(localStorage.getItem('user') || 'null');
}
```
