# Prompt 3.21 — Refactor Barcode Label Print Desktop

## File yang Diubah
`desktop/src/js/barcode.js` (atau bagian print di `products.js`), `desktop/main.js`, `desktop/preload.js`

## Perubahan

```javascript
// SEBELUM — menggunakan printer lokal via Electron
await window.api.barcode.print(productId, quantity, printerName);
const printers = await window.api.barcode.getPrinters();

// SESUDAH — generate label HTML lalu print via window.print()
function printBarcodeLabel(product, quantity) {
    const printWindow = window.open('', '_blank');
    const labels = Array(quantity).fill(null).map(() => `
        <div class="label">
            <div class="product-name">${product.name}</div>
            <svg class="barcode" id="barcode-${product.id}"></svg>
            <div class="price">Rp ${formatCurrency(product.price)}</div>
        </div>
    `).join('');

    printWindow.document.write(`
        <!DOCTYPE html>
        <html>
        <head>
            <title>Print Barcode</title>
            <script src="https://cdn.jsdelivr.net/npm/jsbarcode@3.11.5/dist/JsBarcode.all.min.js"><\/script>
            <style>
                body { margin: 0; }
                .label { display: inline-block; width: 60mm; padding: 4px; border: 1px dashed #ccc; text-align: center; }
                .product-name { font-size: 10px; font-weight: bold; }
                .barcode svg { width: 55mm; height: 20mm; }
                .price { font-size: 12px; }
                @media print { .label { border: none; page-break-inside: avoid; } }
            </style>
        </head>
        <body>
            ${labels}
            <script>
                document.querySelectorAll('.barcode').forEach(el => {
                    JsBarcode(el, '${product.barcode}', { format: 'CODE128', displayValue: true, fontSize: 10 });
                });
                window.onload = () => { window.print(); window.close(); };
            <\/script>
        </body>
        </html>
    `);
    printWindow.document.close();
}
```

## Hapus dari main.js & preload.js
- `barcode:print` handler
- `barcode:getPrinters` handler

## Catatan
- Tidak lagi bergantung pada printer lokal via Electron native API
- Menggunakan `window.print()` — kompatibel dengan Desktop (Electron), Web, dan Android (via dialog print browser)
- Library JsBarcode di-load dari CDN di print window; jika offline gunakan bundled local copy

## Output yang Diharapkan
Cetak label barcode berfungsi di semua platform via browser print dialog.
