# Prompt 0.4 — Setup .gitignore Multi-Platform

## Konteks
Project POS Multi-Platform terdiri dari 4 platform dalam satu repository:
- `desktop/` — Electron + Node.js
- `backend/` — Go GIN
- `web/` — HTML/CSS/JS (nanti)
- `android/` — Capacitor + Android (nanti)

Perlu satu `.gitignore` di root yang mencakup semua platform.

## Tugas

Buat file `.gitignore` di root `Project_POS/` dengan isi berikut:

```gitignore
# ================================
# DESKTOP (Electron + Node.js)
# ================================
desktop/node_modules/
desktop/dist/
desktop/out/
desktop/pos-retail.db
desktop/backups/
desktop/logs/
desktop/.env

# ================================
# BACKEND (Go GIN)
# ================================
backend/logs/
backend/tmp/
backend/.env
backend/config/config_prod.json
backend/config/config_uat.json
backend/config/config_local.json
backend/config/config_bors_kost.json
backend/.scannerwork/

# Go binary & build output
backend/*.exe
backend/*.out
backend/main

# Go test cache
backend/vendor/

# ================================
# WEB
# ================================
web/node_modules/
web/dist/
web/.env
web/.env.local
web/.env.production

# ================================
# ANDROID (Capacitor)
# ================================
android/node_modules/
android/dist/
android/.env
android/android/
android/ios/
android/.gradle/

# APK output
*.apk
*.aab

# ================================
# SHARED
# ================================
shared/.env

# ================================
# OS & EDITOR
# ================================
.DS_Store
.DS_Store?
._*
.Spotlight-V100
.Trashes
ehthumbs.db
Thumbs.db

# VS Code
.vscode/settings.json
.vscode/launch.json
*.code-workspace

# JetBrains
.idea/
*.iml

# ================================
# LOGS & TEMP
# ================================
*.log
*.tmp
*.temp
*.swp
*.swo

# ================================
# SENSITIVE FILES
# ================================
*.pem
*.key
*.cert
*.p12
*.keystore
*-keystore.jks
google-services.json
```

## Output yang Diharapkan
File `.gitignore` di root project yang mencakup semua platform.
Pastikan file sensitif (config prod, .env, keystore) tidak ikut ter-commit.
